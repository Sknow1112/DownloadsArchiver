Downloads Archiver

 IMPORTANT NOTE:     This script has been tested only on MacOS. It may require some modifications to work on other operating systems.

This script is used to automatically archive all files in the current user's Downloads folder that aren't already in a
sub-folder. It creates a new folder with the current date appended to the name "Archive" in the Downloads folder and
moves the files to that folder.

Usage
    [Unzip Folder if Python is not installed (NOT TESTED)]
    Simply click DownloadsArchiver.py

    Make sure you have Python installed on your computer.
    Download the DownloadsArchiver.py script and save it in a convenient location.
    Open a terminal and navigate to the directory where the script is saved.
    Run the script by typing python DownloadsArchiver.py in the terminal and press enter.

Requirements

    Python 3.x
    os library
    shutil library
    datetime library

Note

    This script should be executed with appropriate permissions.
    This script will only archive files that aren't already in a subfolder in the Downloads directory.
